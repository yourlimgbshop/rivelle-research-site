Privacy Policy

Rivelle Research (Personal)

Overview

This is a private, non-commercial personal tool. It uses only the official Etsy Open API v3. No scraping, no browser automation.

Data Collected

  • From Etsy: public listing metadata returned by the Open API (e.g., title, price). No buyer PII is requested.
  • From Users: none. The only user is me, the owner.

Data Use & Retention

  • Data is processed in memory to compute simple aggregates (counts, price stats).
  • No persistent storage. No database. No resale or sharing of data.
  • If OAuth is ever enabled for my own shop, tokens are stored locally and securely on my device only.

Security

  • All API traffic is HTTPS.
  • If OAuth is used, PKCE is implemented; refresh tokens are rotated per Etsy guidance.
