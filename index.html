Rivelle Research (Personal)

Non-commercial research tool for my own Etsy shop. Public data only. No scraping. No browser automation.

What this app is

Rivelle Research (Personal) is a private tool I use to study keywords and public listings on Etsy for my own shop decisions. It uses the official Etsy Open API v3 with OAuth2 + PKCE when needed and always includes the required x-api-key. It never collects buyer personal data and never performs any actions on behalf of other sellers.

Key Features

  • Keyword Research: query public active listings by keyword and aggregate simple metrics (count, price stats).
  • Product Analytics: view public listing metadata for a keyword or a specific shop.
  • Profit Calculator: client-side math only (no API calls).

Compliance & Data Use

  • No scraping or browser automation. Only REST calls to Etsy Open API v3.
  • Public data only for research. No buyer PII, no orders, no messages.
  • Minimal scopes (initially none). OAuth used only if I later access my own shop's non-public data.
  • Zero server storage of API responses. Data is processed in memory and discarded.
